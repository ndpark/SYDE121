1. Compile and run <gps.cpp>
2. Enter the ID of the first point, press Enter
3. Enter the x,y,z coordinates of the point, press Space after each variable, then press Enter after z
4. Pick how accurate the point is, enter either A,B,or C depending on the accuracy. Press Enter.
5. The program will echo the value entered for the first point
6. Repeat steps 2-5 for the second point, press Enter
7. Observe the output which includes : spatial distance, planimetric distance, height difference, and azimuth.
8. Press any key to exit.
